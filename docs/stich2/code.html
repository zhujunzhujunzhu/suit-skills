Suit Skills / 自动化流水线脚本

自动化构建与发布流水线

此技能模块提供了一套完整的 CI/CD 工具集，旨在简化从代码提交到容器化部署的整个流程。通过集成多种安全扫描和性能测试工具，确保每次发布的稳定性。

核心特性

基于 Docker 和 Kubernetes 的云原生架构设计，支持多种编程语言的自动化构建过程：

  • 多阶段并行构建流水线
  • 集成的静态代码质量分析 (SonarQube)
  • 自动化的版本标记与更新规则
  • 支持蓝绿部署与金丝雀发布策略

配置示例

# suit_skill.yaml
pipeline:
  name: production-deploy
  steps:
    - name: checkout
      action: git-pull
    - name: build
      action: docker-build
      params:
        context: ./src
        tag: latest

安装与环境依赖

在开始之前，请确保您的本地环境中已正确配置 Node.js v18+ 以及 Docker Desktop。所有路径配置将在技能安装时自动生成于 .suit/config 目录中。
